# cli-engine Documentation

Developer documentation for `@grekt-labs/cli-engine`.

## What is cli-engine?

The deterministic core logic for grekt CLI, separated for:

- **Testability**: Mock interfaces instead of filesystem
- **Portability**: Runs in CLI, browser, tests, anywhere
- **Determinism**: Same inputs always produce same outputs
